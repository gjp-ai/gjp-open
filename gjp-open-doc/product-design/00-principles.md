# Product Design Principles

## Experience Principles

- Fast content access beats decorative onboarding.
- The first screen should be useful immediately.
- The app should feel like a focused public knowledge library for GJP AI.
- Bilingual behavior must be visible and reliable.
- Media screens should respect mobile habits: swipe, pull-to-refresh, native share/open, and predictable back behavior.

## Visual Direction

- Keep the UI quiet, direct, and content-focused.
- Use native-feeling controls, predictable spacing, and legible typography.
- Avoid large marketing-style hero sections inside the app.
- Prefer stable card dimensions for repeated content.
- Use meaningful icons for tabs and actions.
- Ensure text does not overflow at larger font sizes.
