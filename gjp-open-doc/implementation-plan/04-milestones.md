# Milestones

## Suggested Initial Milestones

| Milestone | Output |
| --- | --- |
| M1 | App opens, navigation works, settings persist, and foundation tests run. |
| M2 | Websites, Q&A, Articles, and Article Detail work against API with unit/API tests. |
| M3 | Images, Videos, Audio, and Files work against API with media smoke tests. |
| M4 | Bilingual, theme, caching, error states, UI automation, and smoke tests complete. |
| M5 | Store-ready builds and metadata complete. |

## First Engineering Task After Approval

Create the mobile repository and implement the foundation:

```text
gjp-open-mobile-expo
```

Initial acceptance criteria:

- App runs on Android and iOS simulators.
- Bottom tab navigation exists.
- Language and theme settings persist.
- API client can fetch `/open/app-settings`.
- Quality commands are documented and runnable.
