# Requirements Overview

## Product Goal

Create a public Android and iOS app for GJP AI that gives users fast, readable access to the same public content currently available on the website:

- Websites
- Q&A
- Articles
- Images
- Videos
- Audio
- Files

The app is public access only. The first release must not require login, account creation, payments, or admin functions.

## Target Users

| User | Need |
| --- | --- |
| Public visitor | Browse curated GJP AI content on mobile. |
| Returning reader | Quickly find recent or tagged articles, questions, sites, and media. |
| Bilingual visitor | Switch between English and Chinese content. |
| Mobile-first user | Open video, audio, image, website, and file links comfortably on a phone. |
