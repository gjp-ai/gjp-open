# Acceptance Criteria

- A user can install and open the app without authentication.
- A user can browse all MVP content sections.
- A user can switch EN/ZH and see matching content.
- A user can open article detail from an article list.
- A user can view images and play video/audio using public API URLs.
- A user sees clear loading, empty, error, and retry states.
- The app passes Android and iOS local smoke tests.
- The app uses only public `/api/open/*` endpoints.
