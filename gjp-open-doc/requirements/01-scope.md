# Requirements Scope

## MVP Scope

The first app release should include:

- Public content browsing for websites, Q&A, articles, images, videos, audio, and files.
- Article list and article detail screens.
- Media preview or playback for image, video, and audio items.
- File open/download handoff to the OS.
- External website and source URL handoff to the system browser.
- Language toggle for `EN` and `ZH`.
- Light and dark themes.
- Theme color selection if it can be implemented without delaying the MVP.
- Search and tag filtering where supported by the API.
- Pull-to-refresh on list screens.
- Infinite scroll or paginated loading.
- Empty, loading, error, and retry states.
- Basic offline resilience through cached last successful responses.

## Out Of Scope For MVP

- User accounts, login, profiles, saved favorites, comments, ratings, or submissions.
- Admin content editing.
- Push notifications.
- In-app purchases.
- Personalized recommendations.
- Full offline media download management.
- Native widgets.
